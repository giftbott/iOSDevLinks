## About
iOS 개발자들이 알아두면 유용한 사이트와 링크를 모아서 각 카테고리별로 정리한 링크 목록입니다. 특히 iOS 개발을 처음 시작하는 사람이 정보 부족으로 힘들어하지 않고 조금이라도 쉽게 접근하는데 도움이 됐으면 합니다.

링크가 깨지거나 다른 문제가 있을 경우 알려주세요. 그리고 추천할 만한 링크는 같이 공유 해주시면 좋겠습니다.
